개인정보처리방침

앱: 주기율표 퀴즈 · 패키지: io.github.depaysement.periodictable · 시행일: 2026-05-13 (v1.1.0)

요약 한 줄 — 퀴즈 풀이 자체는 계정 없이 동작합니다. 리더보드에 참여하는 경우에만 익명 식별자, 직접 입력한 닉네임, 게임 점수가 서버에 저장됩니다. 언제든 앱 내에서 비공개로 전환할 수 있습니다.

1. 수집하는 정보

리더보드 기능을 사용할 때 한해 다음 정보가 수집됩니다.

  • 익명 식별자(Firebase Anonymous UID): 기기·계정과 연결되지 않은 무작위 ID. 같은 사용자임을 식별하기 위해서만 사용됩니다.
  • 닉네임: 사용자가 직접 입력한 표시 이름. 다른 사용자에게 공개됩니다.
  • 게임 점수와 달성 시각: 스피드 모드·세로 본능 모드의 최고 기록.

이름·이메일·전화번호·실명 등의 신원 정보, 위치 정보, 광고 ID, 연락처, 사진 등은 수집하지 않습니다.

2. 수집·이용 시점

사용자가 게임을 마쳤을 때 점수가 일정 기준(이번 달 또는 전체 상위 50위 진입 가능)을 충족하면 앱이 닉네임 등록을 안내합니다. 사용자가 닉네임을 등록한 후에만 정보가 서버로 전송됩니다. 등록 안내를 무시하거나 "나중에"를 선택하면 정보가 전송되지 않습니다.

3. 보관 기간

  • 월간 리더보드 기록: 영구 보관 (지난 달 챔피언 박제 등 기능을 위해).
  • 전체(All Time) 기록: 영구 보관.
  • 사용자가 앱 내에서 "점수 기록을 비공개로 전환"을 선택하면 해당 사용자의 모든 기록이 즉시 삭제됩니다.

4. 기기 내부 저장 항목

다음 항목은 기기 내부에만 저장되며 서버로 전송되지 않습니다.

  • 로컬 최고 기록(스피드 모드 점수, 세로 본능 시간)
  • 해금된 단계, 만점 횟수 등 진행 상태
  • "한글만" 등 사용자 설정값
  • 본인이 신고한 사용자 ID 목록(로컬 차단용)

앱을 삭제하면 이 데이터는 함께 제거됩니다.

5. 인터넷 사용

퀴즈 풀이 자체는 인터넷 없이 동작합니다. 리더보드 조회·점수 제출·닉네임 등록 시에만 인터넷 연결을 사용합니다.

6. 제3자 제공

리더보드 데이터는 Google LLC의 Firebase(Cloud Firestore, Authentication, Analytics) 서비스에 저장·처리됩니다.

  • Firebase 개인정보처리방침: https://firebase.google.com/support/privacy
  • 전송 구간은 HTTPS로 암호화됩니다.

그 외 어떠한 제3자에게도 데이터를 판매·공유하지 않습니다.

7. 광고·분석 도구

광고는 포함되어 있지 않습니다. Firebase Analytics가 활성화되어 있어 앱 설치·세션 같은 기본 사용 통계를 익명으로 수집합니다(개별 식별이 불가능한 집계 데이터).

8. 사용자의 권리

  • 비공개 전환: 앱 내 설정 → 리더보드 → "점수 기록을 비공개로 전환"을 누르면 닉네임과 점수가 즉시 서버에서 삭제됩니다.
  • 닉네임 변경: 같은 메뉴에서 언제든 변경 가능합니다.
  • 리더보드 미사용: 닉네임을 등록하지 않으면 어떤 정보도 서버로 전송되지 않습니다.

9. 어린이 개인정보 보호

본 앱은 만 13세 미만 어린이를 대상으로 한 별도 데이터 수집을 하지 않습니다. 익명 식별자와 닉네임 외 어떠한 신원 정보도 수집하지 않습니다.

10. 정책 변경

본 방침이 변경될 경우 본 페이지에서 변경 일자와 함께 갱신본을 공개합니다.
